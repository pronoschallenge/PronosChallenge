Réglement de PronosChallenge

Frais d'inscription

Pour pronostiquer pendant un an sur PronosChallenge, il est demandé à chaque participant de payer une cotisation de 3 €. Cette somme sera utilisée à la fin du championnat pour offrir des lots aux meilleurs pronostiqueurs !



Classement général

L'attribution des points pour le classement général se fait comme suit :
  • 0 point pour un match non pronostiqué
  • 0 point pour un match pronostiqué non correct
  • 1 point pour un match pronostiqué correctement


Classement Hourra

Le nombre de points attribués pour un pronostic pour ce classement est calculé de la façon suivante :

(1 pt x Nb de parieurs d'un match) / Nb de paris pour un résultat

Si par exemple pour le match Nantes - Lyon, 20 personnes pronostiquent,
  • 16 pour une victoire de Lyon = (1 pts x 20) / 16 soit 1,5 pts par pronostiqueur
  • 3 sur un match nul = (1 pts x 20) / 3 soit 7 pts par pronostiqueur
  • 1 sur une victoire de Nantes = (1 pts x 20) / 1 soit 20 pts pour l'heureux pronostiqueur !
Pour connaître les cotes, reportez vous dans le menu à la rubrique "Pronos" - "Cotes".



Classement Mixte

Le classement mixte est une sorte de moyenne du classement général et du classement hourra. La formule de calcul est la suivante :

Nb de pts mixte = (0.25 x nb points joueur général / nb point 1er général)
+ (0.25 x nb points joueur hourra / nb point 1er hourra)
+ (0.25 x (nb participants - classement joueur général +1) / nb participants)
+ (0.25 x (nb participants - classement joueur hourra +1) / nb participants)
